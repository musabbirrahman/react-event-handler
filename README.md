# Event Planner Landing Page

[![GitHub Repository](https://img.shields.io/badge/GitHub-Repository-blue?logo=github)](https://github.com/musabbirrahman/react-event-handler)

A clean, modern React landing page component for an Event Planning application. This project provides a structured UI to showcase event categories, platform features, user testimonials, and a contact form to help users plan and organize their events effortlessly.

## 🚀 Features Highlighted

The landing page highlights the core capabilities of the Event Planner platform:
* **Easy event creation and management**
* **Customizable event templates**
* **Guest list management**
* **Real-time collaboration**
* **Reminders and notifications**

## 📂 Event Categories Supported

The platform is designed to handle a wide variety of events, neatly categorized into:
* **🎉 Social Events:** Birthday parties, anniversary celebrations, wedding receptions, baby showers, graduation parties, and family reunions.
* **🎭 Entertainment Events:** Concerts, music festivals, film screenings, comedy shows, art exhibitions, and cultural events.
* **🤝 Community Events:** Fundraising events, charity galas, volunteer drives, neighborhood block parties, community festivals, and cultural celebrations.

## 🛠️ Technologies Used

* **React** - Frontend library for building the UI components.
* **CSS3** - For styling the component (`EventPlanner.css`).
